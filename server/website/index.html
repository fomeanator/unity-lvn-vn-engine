The 4-Line Syntax

LVNScript is designed to get out of the way of your storytelling. You can master the syntax rules in under a minute:

1

Labels

:porch

Creates a bookmark/jump destination in your story.

2

Speech & Narration

Mara [smile]: "You came back."

Write dialogue with character names and emotions, or plain text for narration.

3

Interactive Choices

- "I did." -> inside min=2

Present choices to the player, branching directly to other labels with optional stat gates.

4

Engine Operations

bg sprite_url="/content/bg.jpg"

Invoke engine actions like changing backgrounds, camera shakes, or adjusting stats.

Tutorial Lessons

Follow these quick lessons to understand all features and operations of LVNScript.

Lesson 1

Document Structure & Narration

Every story script starts with scene metadata and optional character mappings to map clean dialogue names to internal actor resource IDs. Anything that isn't a command, label, or choice is automatically treated as narration or speech.

scene intro_scene
actor_map Mara=mara_character_id

// This is a comment
This is narration and has no speaker nameplate.
Mara: This is a speech line. Nameplate shows "Mara".
Mara [happy]: I am smiling now! This emits an actor emotion command automatically.
Lesson 2

Branching & Variables

Manage story flow with labels, absolute jumps, and conditional branching. Use set and inc to handle variables, and the if command to branch on conditions.

scene variables_branching
set key="met_guest" value=false
set key="friendship" value=0

:start_story
Mara: Hello! Have we met?
- Yes, we have. -> met_before
- No, this is the first time. -> first_meeting

:met_before
set key="met_guest" value=true
inc key="friendship" by=5
goto check_state

:first_meeting
Mara: Nice to meet you!
goto check_state

:check_state
if expr="friendship >= 5" then="best_buds" else="strangers"

:best_buds
Mara [smile]: We are already great friends!
goto __end

:strangers
Mara: I hope we get to know each other better.
goto __end
Lesson 3

Interactive Gated Choices

Choices are declared using bullet points (-). You can restrict choices using narrative cost values or statistics required by the player. Gated parameters are written directly after the target label.

scene choices_gates
:choices_room
Mara: Do you want to try the forbidden door?

- Enter the door -> enter_room min=5 requires_stat="courage"
- Pay the lockpicker to open it -> open_with_money cost="50 gold"
- Walk away -> leave_room

:enter_room
You courageously break through the door!
goto __end

:open_with_money
The lockpicker opens the door with a click.
goto __end

:leave_room
You walk away safely.
goto __end

Operations Reference

Since LVNScript compiles 1:1 to the LVN engine, operations map directly to engine commands. Below is the list of available commands and their parameters.

Category Command Name Fields Description
Visuals & Camera bg sprite_url (str), id (str) Sets the full-screen background image.
actor id (str), sprite_url (str), show (bool), position (left/center/right), emotion (str) Places, updates, or hides a character on the stage.
fade to (black/white/clear), duration (num) Triggers a full-screen transition fade.
dim alpha (0..1), duration (num) Dims the stage behind dialogue to focus on text.
camera action (shake/zoom/pan), amplitude (num), duration (num) Triggers a camera visual effect.
particles type (rain/snow), on (bool) Toggles background particle effects.
Audio & Timing audio channel (music/sfx/ambient), action (play/stop), url (str) Plays or stops music/sfx.
wait ms (num) Pauses execution for N milliseconds before proceeding.
Flow Control goto label (str) Jumps to a label immediately. (Special label: __end)
if expr (str), then (str), else (str) Conditional jump based on an expression.
call label (str) Jump subroutine (saves return address).
return — Returns from the last call subroutine.
State & Variables set key (str), value (any) Assigns a variable value (string, number, boolean).
inc key (str), by (num) Increments a numeric variable.

LLM System Prompt Context (Markdown) 
# System Instructions for LVNScript (.lvns) Generation

You are an expert story author and coder generating narrative scripts in LVNScript (.lvns) format. Always generate valid syntax matching these precise grammar rules:

## 1. Document Configuration
- **Scene ID**: Define the scene identifier at the top of the file:
  `scene scene_name`
- **Actor Mapping**: Define mappings from user-facing names to asset IDs:
  `actor_map DisplayName=asset_id` (e.g., `actor_map Old Man=old_man`)

## 2. Story Flow & Labels
- **Labels**: Bookmarks/jump destinations are created with a leading colon:
  `:label_id`
- **Gotos & Subroutines**: Jump immediately or use call/return subroutines:
  `goto label_id`
  `call label_id`
  `return` (returns to the last call point)

## 3. Dialogue & Narration
- **Narration**: Any plain text line without a speaker name is narration:
  `The rain began to pour.` (Compiles to say command with who=null)
- **Dialogue**: Character lines follow a speaker name and a colon:
  `Mara: You came back.`
- **Dialogue Emotions**: Define character name, optional emotion in brackets, colon, and text:
  `Mara [smile]: Then come in.` (Emits actor emotion command followed by dialogue)

## 4. Gated & Simple Choices
- **Choices**: Consecutive lines starting with `-` form a choice list. Every choice must specify a target label using `->`:
  `- Choice Text -> target_label`
- **Gated Choices**: You can add optional trailing key-value parameters:
  `- Buy the key -> unlock_door cost="50 gold"`
  `- Break the door -> smash_door min=5 requires_stat="courage"`
  `- Secret choice -> secret_path expr="friendship >= 5"`

## 5. Script Operations
- **Variables**: Assign or increment variables using key-value pairs:
  `set key="flag" value=true`
  `inc key="score" by=1`
- **Environment**: Set background, fade transitions, camera action, or particle layers:
  `bg sprite_url="/content/bg.jpg"`
  `actor id="mara" show=true position="left" sprite_url="/content/mara.png"`
  `fade to="black" duration=0.8`
  `particles type="rain" on=true`
  `camera action="shake" duration=0.5`
  `wait ms=1000`
- **Conditionals**: Jump to different labels based on expression evaluation:
  `if expr="score >= 10" then="victory" else="defeat"`

## 6. Example Script
```lvns
scene chapter_1
actor_map Mara=mara

goto start

:start
bg sprite_url="/content/bg/porch.jpg"
Rain ticked on the porch.
Mara [concerned]: You shouldn't be out in this.

- I did -> warmth_choice
- I'm leaving -> leave

:warmth_choice
inc key="friendship" by=1
Mara [smile]: Then come in.
goto __end

:leave
fade to="black" duration=0.8
goto __end
```
